About Me

Growing up in the Silicon Valley, I quickly became fascinated with the tech scene, in a constant pursuit for more knowledge in the realms of software and web development. At age 10, I started teaching myself how to program, shortly thereafter discovering web design and development. At this time the internet was still evolving and Web 2.0 was just beginning. I am always challenging myself to come up with new, innovative ways to make the web and applications a more user-centered environment. A big way I see this happening is through the emerging User Interface and Experience concentration of visual design, which I've consequently become very engaged in.
